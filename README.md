# نظام طلبات المنتجات

نظام ويب بسيط لإدارة طلبات المنتجات مع صفحة إدارة للأدمن. يعتمد على Firebase Firestore لتجميع الطلبات من جميع الأجهزة مع وجود وضع احتياطي محلي.

## المميزات

- نموذج طلب كامل مع جميع البيانات المطلوبة
- حساب تلقائي للأسعار والعربون
- صفحة إدارة محمية بكلمة مرور لعرض جميع الطلبات
- تخزين مركزي لجميع الطلبات عبر Firebase Firestore
- وضع احتياطي للتخزين المحلي في حال عدم توفر الاتصال بفايربيس
- إمكانية تصدير البيانات كملف JSON

## المنتجات المتاحة

1. الموقد المنزلي - 2600 جنيه
2. دفايه - 2900 جنيه
3. دفاية غاز - 3400 جنيه
4. فرن بلدي - 2400 جنيه

## التشغيل

1. افتح ملف `public/index.html` في المتصفح مباشرة
2. أو استخدم أي خادم محلي بسيط (مثل Live Server في VS Code)
3. فعّل Firestore من مشروعك على Firebase ثم حدّث ملف `public/firebase-config.js` ببيانات مشروعك (الكود الافتراضي مجرد عينات)

## صفحات الموقع

- **صفحة الطلب**: `public/index.html`
- **صفحة الإدارة**: `public/admin.html`

## إعداد فايربيس

1. من وحدة تحكم Firebase انسخ إعدادات تطبيق الويب (`apiKey`, `authDomain`, ... إلخ).
2. افتح الملف `public/firebase-config.js` واستبدل القيم الافتراضية بالقيم الحقيقية، أو عرّف الكائن `window.FIREBASE_CONFIG` قبل تحميل الملف.
3. تأكد من تفعيل Firestore وقواعد الأمان المناسبة (`orders` و `deletedOrders`).
4. انشر التعديلات باستخدام `firebase deploy --only hosting` ليتم تحميل مكتبات فايربيس الجديدة.

عند عدم توفر الاتصال بفايربيس سيعود التطبيق تلقائياً لاستخدام التخزين المحلي لضمان استمرار العمل.

## كلمة مرور الإدارة

كلمة المرور الافتراضية: `admin123`

يمكن تغييرها في ملف `public/admin.js` في المتغير `ADMIN_PASSWORD`

## ملاحظات

- يتم دفع عربون 200 جنيه قبل الاستلام
- طريقة الدفع: فودافون كاش فقط
- جميع الطلبات يتم حفظها في Firestore (مع نسخة احتياطية محلية إذا تعذر الاتصال)
- يمكن تصدير البيانات من صفحة الإدارة كملف JSON
- البيانات محفوظة محلياً في المتصفح فقط
