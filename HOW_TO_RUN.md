# كيفية تشغيل الموقع الجديد (Next.js)

لقد قمت بتحويل المشروع من صفحات **HTML** عادية إلى مشروع **Next.js** حديث ومنظم. لهذا السبب، لن ترى التغييرات بمجرد فتح ملفات `.html` القديمة.

## الخطوات لتشغيل الموقع ورؤية التغييرات:

1. **تثبيت المكتبات الجديدة**:
   افتح الـ Terminal في المجلد واكتب:
   ```bash
   npm install
   ```

2. **تشغيل الموقع في وضع التطوير**:
   بعد التثبيت، اكتب:
   ```bash
   npm run dev
   ```

3. **فتح الموقع**:
   سيظهر لك رابط (غالباً `http://localhost:3000`). افتحه في المتصفح.

## الصفحات الجديدة:
- **الصفحة الرئيسية للعملاء**: `http://localhost:3000/`
- **لوحة الأدمن (الجديدة)**: `http://localhost:3000/admin/dashboard`
- **إدارة الطلبات**: `http://localhost:3000/admin/orders`
- **إدارة المنتجات**: `http://localhost:3000/admin/products`

> [!NOTE]
> الملفات القديمة مثل `admin.html` و `index.html` لا تزال موجودة ولكنها لم تعد تُستخدم. التعديلات الجديدة تمت برمجتها داخل مجلد `src/app`.

**إذا واجهت أي مشكلة في التشغيل، أنا موجود للمساعدة!**
